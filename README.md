# Guide Market Shopify Theme Branch

This branch is the primary Shopify theme branch for Guide Market.

The Next.js app remains on the `main` branch as a separate build, but Shopify Themes cannot use a Next.js app directly. Use this `shopify-theme` branch when connecting the store in Shopify.

Included Shopify theme areas:

- Premium coming-soon home page
- Empty shop template at `/collections/all`
- About template
- FAQ template
- Contact template with Shopify contact form
- Privacy Policy template
- Terms of Service template
- Refund Policy template
- Empty Top Sells, New Arrivals, Reviews, and Launch Updates sections

No products, prices, cart logic, checkout buttons, fake reviews, or fake catalog data are included.
